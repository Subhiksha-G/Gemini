# My Professional Portfolio 🚀

This is a multi-page portfolio website built to showccase my skills in Frontend Development.

## Features
- **Dynamic Team Directory**: Fetches real time data from the RandomUser API.
- **Live Search**: Filter through team member instantly.
- **Dark / Light Mode**: User-preference theme toggling with LocalStorage memory.
- **Interactive Hero**: Features a custom-built typewriter animation and parallax effects.

## Tech Stacks
- HTML5, CSS3 (Flexbox, Grid)
- Vanilla JavaScript (Async/Await, DOM Manipulation)
- GitHub & Netlify (Continous Deployment)
